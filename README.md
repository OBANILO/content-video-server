# Content Video Server - Fresh Pexels B-roll Fixed

Fixes:
- Uses multiple exact Pexels search phrases, including the user's own search phrases
- Downloads up to 32 unique Pexels videos
- Avoids duplicate Pexels video IDs and links
- Cuts clips into 6-second unique b-roll parts so the video feels fresh
- Repeats only as last fallback if Pexels does not provide enough clips
- Forces ElevenLabs voiceover as final audio
- Removes top title overlay and website watermark

Deploy on Render, then generate a fresh video.
